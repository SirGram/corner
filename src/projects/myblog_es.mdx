## Objetivo

---

<br />
Esta web fue un ejercicio para aprender un poco del stack MERN. Nest es un framework
basado en Express.js que es muy fácil de configurar y poner en marcha.

 Con este proyecto,
he creado una aplicación CRUD que tiene varias rutas, permisos de usuario diferentes
y utiliza guards para limitar el acceso a los usuarios externos.

<br />
## Características

---

<br />
-Operaciones CRUD para posts y comentarios. 
-Autenticación de usuarios conJWT y guards básicos de Nest. 
-Posts MDX. 
-Barra de búsqueda. 
-Edición y eliminación
de perfiles de usuario. 
-Resaltado de código. 
-Documentación de Swagger. 
-Paginación.
-ShadCN.
